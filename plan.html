Five lanes D0a gates everything else

D0a · fast directional probe

Short ~80–120 step fine-tune

Day-1 signal. Drives every gating decision below.
Hypothesis
A short FT can tell us in hours whether PL LDDT moves the right way and whether IP LDDT collapses — without burning 20 h.
Change
Reference hyperparams (LR 3e-4, EMA 0.99, warmup 50) but only ~80–120 steps and dense checkpoints (num_gradient_steps_per_epoch: 4).
Measure
PL LDDT & IP LDDT on the 17 held-out at every checkpoint.
Worked if
PL LDDT trend is up, IP LDDT doesn't fall.
Falsified if
PL LDDT flat / IP LDDT degrades by >1 lDDT — re-route per gates.
Cost: ~3–6 h on the H100 · Config: configs/d0a_short.json
D0b · unattended reference

Reproduce the published ~350-step run

Anchors our numbers to Apheris. Does not gate D1 or any pivot.
Hypothesis
The published recipe gives a directionally correct reference checkpoint to compare against.
Change
Nothing vs the published protocol — LR 3e-4, EMA 0.99, ~350 steps, no templates, baseline MSAs.
Measure
Same as D0a, plus DockQ on the PL interface for the final ckpt.
Worked if
PL LDDT > base on at least 10/17 eval complexes.
Falsified if
No PL LDDT gain anywhere on the curve → premise (10 structures help) fails for the reference recipe.
Cost: ~20 h, unattended · Config: configs/reference.json
D1 · regularization map

EMA × LR grid around reference

Cheap, characterises the stability window of the recipe.
Hypothesis
The published EMA=0.99 / LR=3e-4 point is unlikely to be optimal for this exact split — a small grid will reveal a sweet spot or robustness.
Change
Hold seeds, warmup, crop fixed; vary ema_decay ∈ {0.95, 0.995} and learning_rate ∈ {1e-4, 6e-4}.
Measure
PL LDDT & IP LDDT curves per cell, side-by-side.
Worked if
Any cell beats reference on PL LDDT without losing >1 lDDT on IP.
Falsified if
All cells track within noise → recipe is robust, data is the bottleneck (pivot to D2).
Cost: 3–4 runs × ~8 h ≈ 24–32 h · Configs: configs/d1_*.json
D2 · similarity-based augmentation

+20–40 retrieved similar complexes

High upside. Matches organizers' graph-similarity hint.
Hypothesis
10 is too few; adding chemotypically or structurally similar liganded complexes will improve transfer more than any hyperparam tweak.
Change
Retrieve ~20–40 augmentation complexes (protein-side via Foldseek/PDE family; ligand-side via Tanimoto + the rest of the XChem 5S* fragments) and mix 2:1 with the original 10.
Measure
Per-eval-complex PL LDDT vs ligand-similarity-to-train.
Worked if
Beats the best D1 cell AND gains are larger on chemotypically distant eval points.
Falsified if
Only helps similar-to-augmentation points and hurts the distant ones — publishable negative result on sample efficiency.
Real bottleneck is MSA generation for the augmentation complexes, not training. Start that pipeline on the VM's CPU resources as soon as D0a's signal looks promising.
Train cost: ~15–20 h · Config: configs/d2_augmented.json
D3 · generalisation guard

Pareto plot, free on every checkpoint

Always-on safety lane. No new training cost.
Hypothesis
Fine-tuning trades general accuracy for PDE10A accuracy somewhere; we want the curve, not a binary claim.
Change
Nothing — every D0/D1/D2 checkpoint is also evaluated on the no-regression proxy.
Day-1 proxy
IP LDDT on the held-out 17. No external set needed for the day-1 picture.
Day-2 proxy
Add a 5–8 complex non-PDE10A PoseBusters-style set. Pulling these in costs an MSA pass each — defer until pipeline is solid.
Worked if
One checkpoint is strictly Pareto-better than base OF3 on (PL LDDT, IP LDDT delta).
Cost: ~free · Output: a Pareto plot for the presentation.

Metrics legend what each number means

Pose quality · headline

PL LDDT

ApherisFold's lddt_inter_protein_ligand — literature LDDT-PLI. Scores ligand–protein interface contacts in [0,1] with a post-CASP15 penalty for false contacts. Default monitored metric.

Protein quality · safety

IP LDDT

Intra-protein LDDT. Day-1 no-regression proxy — if this drops on the 17 held-out, the model has degraded on the protein side itself.

Interface · monomer caveat

ICP LDDT

Inter-chain protein LDDT. Likely 0 or NaN on monomeric PDE10A — confirm on first run, don't read it as a signal otherwise.

Pose · presentation

Pose RMSD & DockQ

Headline numbers for the deck. RMSD is noisy on small fragments — use PL LDDT for early stopping, RMSD/DockQ for storytelling.
